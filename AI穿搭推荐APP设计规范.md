# AI 穿搭推荐 APP 设计规范

## 1. 视觉定位

整体风格为「轻量、干净、柔和、AI 感」。界面以白色和浅灰为底，紫色作为品牌主色，辅以浅紫渐变、细描边、柔和阴影，营造精致的女性向智能穿搭工具感。

- 品牌名：`clozAi`
- 关键词：AI、穿搭、温柔、清爽、轻奢、日常实用

## 2. 画布与布局

推荐移动端基准尺寸：`375 x 812px` 或 `390 x 844px`。

页面规则：

- 页面背景：`#F7F7FB` / `#FAFAFC`
- 内容主区域左右边距：`16px`
- 卡片圆角：`12px - 16px`
- 底部导航高度：`64px - 72px`
- 顶部安全区：保留 `44px`
- 页面主按钮距底部导航或安全区：`16px - 24px`
- 卡片间距：`12px`
- 分组间距：`20px - 28px`

整体布局以纵向滚动为主，局部横向卡片列表用于今日推荐、灵感、收藏等模块。

```text

Secondary: #8A8A99
```

### 背景与线条

```text

Divider: #ECECF2

```

### 状态色

```text
Like / Favorite: #FF5C8A

Black CTA: #111217
```

### 按钮渐变

```css

```

## 4. 字体规范

建议使用系统字体：

```css
font-family: -apple-system, BlinkMacSystemFont, "PingFang SC", "Microsoft YaHei", sans-serif;
```

字号层级：

```text
Logo / Brand: 24px / 28px, Medium
页面标题: 18px / 24px, Semibold
模块标题: 16px / 22px, Semibold
正文: 14px / 20px, Regular
辅助文字: 12px / 18px, Regular
标签文字: 11px - 12px
底部导航: 10px - 11px
数据强调: 24px - 28px, Semibold
```

文字整体不要过粗，保持轻盈感。数字如 `92%`、`168cm` 可使用中等字重突出。

## 5. 核心组件

### 主按钮

- 高度：`48px`
- 圆角：`24px`
- 背景：紫色渐变
- 文字：白色，`14px - 15px`
- 用于：下一步、生成搭配、一键购买、分享

### 次级按钮

- 高度：`40px - 44px`
- 背景：白色
- 描边：`#E6E3F2`
- 文字：主紫色或深灰
- 用于：保存、重新推荐、上传照片

### 标签 / Chip

- 高度：`28px - 32px`
- 圆角：`14px - 16px`
- 选中态：紫色渐变或浅紫底 + 紫色文字
- 未选中：白底 + 浅灰描边
- 用于：通勤、约会、旅行、日常休闲、轻熟优雅等

### 输入框

- 高度：`44px - 48px`
- 圆角：`12px`
- 背景：`#FFFFFF`
- 描边：弱化或无描边
- Placeholder：`#B8B8C5`

### 卡片

- 背景：白色
- 圆角：`14px - 16px`
- 描边：`#ECECF2`
- 阴影：非常轻

```css
box-shadow: 0 8px 24px rgba(80, 72, 120, 0.06);
```

## 6. 导航规范

底部导航包含 4 个入口：

```text
首页 / 灵感 / 衣橱 / 我的
```

图标风格：

- 线性图标
- 选中态使用紫色
- 未选中态使用浅灰
- 图标尺寸：`20px - 22px`
- 文案字号：`10px - 11px`

顶部导航：

- 左侧返回
- 中间标题
- 右侧操作，如搜索、更多、收藏、分享、刷新
- 图标保持细线风格，不使用厚重实心图标

## 7. 页面规范

### 启动页

- 白色或极浅紫背景
- 中心展示 `clozAi` logo
- 使用浅紫星光、弧线点缀
- 底部黑色主按钮「开始体验」
- 整体留白充足，强调品牌感

### 登录页

- 顶部品牌 logo
- 中部人物模特图
- 手机号输入框
- 紫色主按钮「获取验证码」
- 微信登录为白底描边按钮
- 协议文字置底，小字号弱化

### 风格目标设置页

- 顶部问题式标题
- 多组选项标签
- 身高体重使用数值步进器
- 肤色使用圆形色卡
- 底部固定主按钮

### 上传与分析页

- 双卡片上传正面照 / 侧面照
- 上传成功后展示人物预览
- 分析进度使用勾选列表 + 紫色进度条
- 页面保持安静，突出 AI 分析状态

### 特征分析页

- 左侧或中部人体轮廓
- 右侧列出身高、体型、肩宽、腰臀比、发色等
- 适合风格以 chip 展示
- 底部主按钮「查看推荐穿搭」

### 首页推荐页

- 顶部问候 + 今日推荐问题
- 特征分析卡展示匹配度
- 今日推荐用横向卡片
- 卡片图片为主，文案极简
- 底部导航固定

### 灵感页

- 双列瀑布流
- 顶部分类标签横滑
- 图片圆角 `10px - 12px`
- 展示点赞数、收藏状态
- 视觉重点在穿搭图片

### 衣橱页

- 空状态使用浅紫插画
- 分类标签：上衣、外套、裤装、裙装、鞋包
- 空状态按钮「添加单品」
- 有内容时使用网格或列表展示

### 单品详情页

- 大图居中展示单品
- 标题 + 风格标签
- 属性表格：颜色、材质、适合季节、购买时间
- 底部主按钮「生成搭配」

### 搭配详情页

- 顶部显示适配度，如 `92%`
- 中间大图展示整套穿搭
- 右侧悬浮工具：换背景、换姿势、3D 预览
- 下方为穿搭亮点说明
- 底部主按钮「一键购买整套」

### AI 试穿页

- 全屏试穿图为核心
- 顶部返回、刷新、分享
- 右侧浮动对比按钮
- 底部展示已选搭配缩略图
- 底部操作：保存、分享、下载

### 我的页

- 顶部头像 + 昵称
- 紫色会员卡片
- 设置列表使用细分割线
- 保持简洁，不做复杂装饰

## 8. 图片与插画规范

图片应以真实穿搭模特图为主，背景干净，色调偏暖白、米白、浅灰。人物需完整展示服装轮廓，避免过暗、强滤镜、复杂背景。

插画仅用于空状态、启动页装饰、AI 分析状态，风格为浅紫线性插画，不抢主视觉。

## 9. 动效建议

- 页面切换：轻微淡入 + 上移 `8px`
- 按钮点击：`scale(0.98)`
- AI 分析：进度条平滑推进
- 收藏：心形图标轻微弹跳
- 图片卡片加载：骨架屏，不使用强 loading 动效

## 10. 设计关键词总结

```text
白底、浅灰层次、紫色渐变、圆角卡片、真实穿搭图片、轻量图标、低饱和阴影、女性向、AI 智能感、清爽留白
```
